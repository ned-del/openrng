How It Works

Three steps. Zero trust required.

You don't need to understand cryptography. You just need to know: no one can cheat.

01

🔐 Numbers are committed before you ask

A time-locked proof (VDF) + external randomness (drand beacon) generate each result. No single party — not even us — can predict the outcome.

Technical: Verifiable Delay Function ensures sequential computation that can't be parallelized. drand provides distributed threshold randomness.
02

📋 Every result gets a receipt

Results are batched into a Merkle tree — change one result and the entire tree breaks. Each roll gets a unique cryptographic proof you can check.

Technical: Merkle tree leaves contain individual proofs. Tamper with one leaf and the root hash changes, invalidating the entire batch.
03

⛓️ Receipts are anchored on-chain

Merkle roots are published to Polygon — permanent, public, immutable. Anyone can verify any result, forever.

Currently on Amoy Testnet. Mainnet deployment planned. All anchoring is publicly verifiable on PolygonScan.

Trust

Most "random" bots are black boxes

You trust their server isn't rigged. That's not verification — that's faith.

⚠️ Standard bots

  • Server-side Math.random() — completely unverifiable
  • Operator can see, predict, or manipulate results
  • No audit trail — results can be silently changed
  • "Trust me" is the only guarantee

🛡️ OpenRNG

  • Time-locked VDF — can't be pre-computed or rushed
  • External drand beacon — no single point of trust
  • Merkle proofs — every result independently verifiable
  • Polygon anchoring — permanent, immutable audit trail

Developers

Build with verified randomness

Integrate OpenRNG into your own apps, games, and bots. TypeScript-first, open source.

TypeScript @openrng/sdk
import { OpenRNG } from '@openrng/sdk';

const rng = new OpenRNG({
  network: 'polygon-amoy',
  verify: true
});

// Generate a verifiable random number
const result = await rng.generate({
  type: 'dice',
  sides: 20,
  count: 2
});

console.log(result.values);  // [14, 7]
console.log(result.proof);   // { leafHash, merkleRoot, ... }

// Independently verify any result
const valid = await rng.verify(result.proof);
console.log(valid); // true ✅
