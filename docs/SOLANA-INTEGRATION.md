# OpenRNG — Solana Integration Plan

## Why Solana

**Pay.sh** (Solana Foundation + Google Cloud, launched May 5 2026) enables AI agents to autonomously discover, pay for, and consume APIs using stablecoins on Solana. Google calls 2026 "the year of the agentic enterprise."

**Solana Agent Kit** (SendAI) provides 60+ pre-built actions for AI agents on Solana — but no verifiable RNG action.

OpenRNG on Solana = verifiable randomness as a Pay.sh service + Solana Agent Kit action. AI agents can find us, pay us, and use us without human intervention.

## Solana Program (Anchor Framework)

```rust
// programs/merkle_anchor/src/lib.rs

use anchor_lang::prelude::*;

declare_id!("OpenRNG111111111111111111111111111111111111");

#[program]
pub mod merkle_anchor {
    use super::*;

    pub fn initialize(ctx: Context<Initialize>) -> Result<()> {
        let state = &mut ctx.accounts.state;
        state.authority = ctx.accounts.authority.key();
        state.total_batches = 0;
        state.total_tokens = 0;
        Ok(())
    }

    pub fn anchor_batch(
        ctx: Context<AnchorBatch>,
        batch_id: String,
        merkle_root: [u8; 32],
        batch_size: u64,
        client_id: String,
    ) -> Result<()> {
        let batch = &mut ctx.accounts.batch;
        let state = &mut ctx.accounts.state;
        let clock = Clock::get()?;

        batch.batch_id = batch_id;
        batch.merkle_root = merkle_root;
        batch.batch_size = batch_size;
        batch.client_id = client_id;
        batch.block_time = clock.unix_timestamp;
        batch.slot = clock.slot;

        state.total_batches += 1;
        state.total_tokens += batch_size;

        emit!(BatchAnchored {
            batch_id: batch.batch_id.clone(),
            merkle_root,
            batch_size,
            slot: clock.slot,
        });

        Ok(())
    }
}

#[derive(Accounts)]
pub struct Initialize<'info> {
    #[account(init, payer = authority, space = 8 + 32 + 8 + 8)]
    pub state: Account<'info, ProgramState>,
    #[account(mut)]
    pub authority: Signer<'info>,
    pub system_program: Program<'info, System>,
}

#[derive(Accounts)]
#[instruction(batch_id: String)]
pub struct AnchorBatch<'info> {
    #[account(
        init,
        payer = authority,
        space = 8 + 64 + 32 + 8 + 64 + 8 + 8,
        seeds = [b"batch", batch_id.as_bytes()],
        bump,
    )]
    pub batch: Account<'info, BatchRecord>,
    #[account(mut, has_one = authority)]
    pub state: Account<'info, ProgramState>,
    #[account(mut)]
    pub authority: Signer<'info>,
    pub system_program: Program<'info, System>,
}

#[account]
pub struct ProgramState {
    pub authority: Pubkey,
    pub total_batches: u64,
    pub total_tokens: u64,
}

#[account]
pub struct BatchRecord {
    pub batch_id: String,
    pub merkle_root: [u8; 32],
    pub batch_size: u64,
    pub client_id: String,
    pub block_time: i64,
    pub slot: u64,
}

#[event]
pub struct BatchAnchored {
    pub batch_id: String,
    pub merkle_root: [u8; 32],
    pub batch_size: u64,
    pub slot: u64,
}
```

## TypeScript Anchor Client

```typescript
// src/blockchain/solana-anchor.ts

import { Connection, PublicKey, Keypair } from '@solana/web3.js';
import { AnchorProvider, Program, Wallet } from '@coral-xyz/anchor';
import { logger } from '../utils/logger';

export class SolanaAnchor {
  private connection: Connection;
  private program: Program;
  private authority: Keypair;

  constructor(config: {
    rpcUrl: string;         // e.g. https://api.devnet.solana.com
    programId: string;
    privateKey: string;     // base58 or byte array
  }) {
    this.connection = new Connection(config.rpcUrl, 'confirmed');
    this.authority = Keypair.fromSecretKey(/* decode privateKey */);

    const wallet = new Wallet(this.authority);
    const provider = new AnchorProvider(this.connection, wallet, {
      commitment: 'confirmed',
    });

    // Load program IDL (generated by Anchor build)
    this.program = new Program(IDL, new PublicKey(config.programId), provider);
  }

  async anchorBatch(params: {
    batchId: string;
    merkleRoot: string;
    batchSize: number;
    clientId: string;
  }): Promise<{ txHash: string; slot: number }> {
    const rootBytes = Buffer.from(params.merkleRoot, 'hex');

    // PDA for the batch record
    const [batchPda] = PublicKey.findProgramAddressSync(
      [Buffer.from('batch'), Buffer.from(params.batchId)],
      this.program.programId,
    );

    const tx = await this.program.methods
      .anchorBatch(
        params.batchId,
        Array.from(rootBytes),
        params.batchSize,
        params.clientId,
      )
      .accounts({ batch: batchPda })
      .rpc();

    logger.info(`Solana anchor: ${params.batchId} tx=${tx}`);
    return { txHash: tx, slot: 0 }; // slot from confirmation
  }

  async verifyBatch(batchId: string): Promise<{
    exists: boolean;
    merkleRoot: string;
    slot: number;
  }> {
    const [batchPda] = PublicKey.findProgramAddressSync(
      [Buffer.from('batch'), Buffer.from(batchId)],
      this.program.programId,
    );

    try {
      const batch = await this.program.account.batchRecord.fetch(batchPda);
      return {
        exists: true,
        merkleRoot: Buffer.from(batch.merkleRoot).toString('hex'),
        slot: batch.slot.toNumber(),
      };
    } catch {
      return { exists: false, merkleRoot: '', slot: 0 };
    }
  }
}
```

## Dual-Chain Anchoring

Add to `src/blockchain/anchor.ts`:

```typescript
export class MultiChainAnchor {
  private polygon: PolygonAnchor;
  private solana?: SolanaAnchor;

  constructor(polygon: PolygonAnchor, solana?: SolanaAnchor) {
    this.polygon = polygon;
    this.solana = solana;
  }

  async anchorBatch(params: AnchorParams): Promise<MultiAnchorResult> {
    // Anchor to both chains in parallel
    const results = await Promise.allSettled([
      this.polygon.anchorBatch(params),
      this.solana?.anchorBatch(params),
    ]);

    return {
      polygon: results[0].status === 'fulfilled' ? results[0].value : null,
      solana: results[1]?.status === 'fulfilled' ? results[1].value : null,
    };
  }
}
```

```json
    "solscan": "https://solscan.io/tx/..."
  }
}
```

## Pay.sh Integration

Pay.sh lets AI agents pay for APIs with USDC on Solana.

### Registration

Register OpenRNG as a Pay.sh service:

```json
{
  "service": "openrng",
  "name": "OpenRNG — Verifiable Randomness",
  "description": "Provably fair random numbers with on-chain Merkle proofs",
  "endpoint": "https://api.openrng.io",
  "pricing": {
    "free_tier": { "requests_per_minute": 100, "cost": 0 },
    "pro_tier": { "requests_per_minute": 10000, "cost_usdc_monthly": 19 },
    "per_request": { "cost_usdc": 0.0001 }
  },
  "payment_address": "<OpenRNG Solana wallet>",
  "supported_tokens": ["USDC", "SOL"]
}
```

### Agent Payment Flow

```
AI Agent (Google Cloud / Gemini / Claude Code)
    │
    ├── Discovers OpenRNG via Pay.sh marketplace
    ├── Agent's Solana wallet has USDC balance
    ├── Pay.sh handles: auth token issuance + USDC transfer
    │
    ▼
POST https://api.openrng.io/v1/tokens/request
  Header: x-api-key: <pay.sh-issued-token>
  Header: x-payment-tx: <solana-tx-hash>
    │
    ▼
OpenRNG verifies payment on Solana → serves tokens
```

### Implementation

1. Add Pay.sh webhook handler to verify Solana payment transactions
2. Issue temporary API keys tied to payment confirmations
3. Track usage against pre-paid USDC balance
4. Auto-expire keys when balance runs out

## Solana Agent Kit Action

Register as a SendAI action:

```typescript
// solana-agent-kit action
{
  name: "openrng_random_number",
  description: "Generate a provably fair random number with blockchain verification",
  parameters: {
    min: { type: "number", description: "Minimum value" },
    max: { type: "number", description: "Maximum value" },
    quantity: { type: "number", description: "How many numbers", default: 1 },
  },
  execute: async (params, context) => {
    const rng = new OpenRNG({
      agentId: context.agent.id,
      endpoint: 'https://api.openrng.io',
      apiKey: context.paysh.apiKey, // from Pay.sh
    });
    return await rng.number({ min: params.min, max: params.max });
  }
}
```

## Decision: When to Build

**Trigger:** When any of these happen:
1. First paying customer asks for Solana
2. Pay.sh opens to third-party API registration
3. Solana Agent Kit adds a "marketplace" for tool discovery
4. You have 10+ active users on Polygon and want to expand reach

**Do not build before you have users on Polygon.** The Solana work is a growth lever, not a prerequisite.
