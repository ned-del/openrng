第一層：結果不可篡改

Patent #1 — Verifiable Randomness · 已核准
🎮 玩家的擔心：「我明明快要抽中了，遊戲公司是不是在後台偷偷把我的卡換掉了？」
✅ OpenRNG 的保證：所有隨機數在產生的那一刻，就被鎖進一棵「密碼學大樹」（Merkle Tree），根部的指紋被永久刻在公開的區塊鏈上。一旦刻上去，就算是遊戲公司的老闆親自下令，也改不了你的任何一張卡。改了？數學會立刻暴露。

Every random value is locked in a Merkle tree and anchored to a public blockchain. After commitment, modifying any single value is mathematically detectable — by anyone.

第二層：沒人能偷看

Patent #2 — Dual Temporal Integrity · 申請中
🎮 玩家的擔心：「就算結果改不了，但遊戲公司的工程師會不會在『刻上去之前』就偷看了？看到這批有大獎就不用，換一批沒大獎的刻上去？」
✅ OpenRNG 的保證：系統引入了「時間鎖」（VDF）。每次產生新的一批隨機數時，系統必須強制運算 4 秒鐘。在這 4 秒鐘內，世界上沒有任何人——包括 GM、工程師、公司老闆、甚至駭客——能看到結果。等時間一到，結果直接鎖定，想挑也來不及了。

A Verifiable Delay Function (VDF) creates a mandatory time barrier. During computation, nobody — not even the operator — can preview the results. When it completes, the values are locked. Too late to cherry-pick.

兩層加起來的效果：

🔒 鎖上去之前：沒人看得到（時間鎖保證）
🔒 鎖上去之後：沒人改得了（密碼學保證）

Before commitment: values are UNKNOWABLE. After commitment: values are UNALTERABLE. This is Dual Temporal Integrity.

第三層：你自己就能查

Patent #3 — Portable Trust Assertion · 申請中
🎮 玩家的擔心：「你說的這些我怎麼知道是真的？難道要我自己去讀區塊鏈？」
✅ OpenRNG 的保證：每次抽卡結果都會附帶一張「數位收據」（VEO）。這張收據記錄了：隨機數從哪裡來（多個獨立來源）、品質評分多少（0-1000 分，AAA 到 LOW）、誰簽的名、有沒有上鏈。你可以拿著這張收據去 verify.openrng.io 自己驗——不需要問遊戲公司，不需要連遊戲伺服器，甚至可以離線驗。

Every outcome comes with a Verifiable Entropy Object (VEO) — a digital receipt carrying provenance, quality score, cryptographic signature, and blockchain anchor. Any player can verify it at verify.openrng.io — independently, offline, without contacting the game server.
