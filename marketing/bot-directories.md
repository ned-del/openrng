## BotoStore (botostore.com)

**Name:** OpenRNG Dice Bot
**Username:** @OpenRNG_Dice_Bot
**Category:** Entertainment / Games
**Short Description:** Provably fair dice rolls, coin flips, raffles, and giveaways — every result has a cryptographic proof you can check.

**Full Description:**

OpenRNG Dice Bot generates verifiable random numbers using Verifiable Delay Functions (VDF), drand public randomness, and Merkle tree proofs anchored on Polygon Amoy (testnet).

Unlike regular dice bots, every result is pre-committed before you ask — the bot reveals values that were already locked in. You can verify any result on-chain.

**Commands:**
- /roll — Roll any dice (e.g., /roll 2d20)
- /flip — Flip a coin
- /pick — Pick from options
- /shuffle — Randomize a list
- /raffle — Run a fair giveaway
- /lottery — Draw numbers
- /wheel — Spin a weighted wheel
- /teams — Split into teams
- /draw — Random draw
- /verify — Check cryptographic proof
- /about — How it works

Works inline in any chat: @OpenRNG_Dice_Bot roll 1d20

Free. Open source (code release in progress). No ads, no tracking.

**Tags:** dice, random number generator, provably fair, raffle, giveaway, games
